# Game Icons

Place resource and upgrade icons here (e.g. `gold.png`, `souls.png`).

For Unity to load them at runtime, copy or symlink this folder to:
`Assets/Resources/Game/SampleIdleGame/Art/icons/`

Icons are referenced in `resources.json` and `upgrades.json` via `iconPath` (e.g. `"Art/icons/gold"`).
