# SampleIdleGame

Reference game for the engine. Duplicate this folder to create a new game.

## Structure

- `Definitions/` - game.json, theme config
- `Content/` - resources, production, upgrades, quests
- `UI/` - layouts and templates
- `Localization/` - translation files (en.json, pt-BR.json, etc.)
- `Art/` - sprites, fonts, audio
