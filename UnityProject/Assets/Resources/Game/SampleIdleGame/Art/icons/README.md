# SampleIdleGame Icons

Place PNG icons here for Resources.Load to find them at runtime.

Expected files (from resources.json / upgrades.json):
- gold.png
- souls.png
- upgrade_gold_1.png
- upgrade_gold_2.png

Path format: `Art/icons/<name>` in config maps to `Game/SampleIdleGame/Art/icons/<name>` for Resources.Load.
